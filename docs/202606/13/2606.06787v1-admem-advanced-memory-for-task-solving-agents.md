## 摘要
大型语言模型（LLMs）作为使用工具的主体显示出潜力，但在需要记忆、组织和重用知识的长期任务中仍存在局限。先前的记忆方法旨在解决这一问题，但主要侧重于存储事实信息。最近关于程序记忆的工作改进了任务重用，但往往简化为重放过去的成功，而没有处理失败案例或在线可扩展性。我们引入了一个统一且自动化的记忆框架，该框架在结合短期和长期存储的双层设计中集成了语义、情景和程序记忆。一个包含行动者、记忆和评论者智能体的多智能体架构实现了自动记忆生成、奖励标注和自适应检索。长期记忆通过基于奖励的评估、合并和剪枝进行管理，确保可扩展性和持续改进。跨多种环境的实验表明，与现有基线相比，我们的方法在长期多轮任务上提高了鲁棒性和成功率。这项工作强调了全面、自适应记忆对于推进基于LLM的智能体的重要性。

## Abstract
Large Language Models (LLMs) show promise as tool-using agents but remain limited in long-horizon tasks that require remembering, organizing, and reusing knowledge. Prior memory approaches aim to resolve the situation, but mainly focus on storing factual information. Recent work on procedural memory improves task reuse, yet often reduces to replaying past successes without addressing failure cases or online scalability. We introduce a unified and automatic memory framework that integrates semantic, episodic, and procedural memory in a bi-level design combining short-term and long-term stores. A multi-agent architecture with actor, memory, and critic agents enables automatic memory generation, reward annotation, and adaptive retrieval. Long-term memory is managed through reward-based evaluation, merging, and pruning, ensuring scalability and continual improvement. Experiments across various environments show that our approach improves robustness and success on long multi-turn tasks compared to existing baselines. This work highlights the importance of comprehensive, adaptive memory for advancing LLM-based agents.

## 2. 方法论：核心思想、关键技术细节

### 核心思想
- 提出一个统一且自动化的记忆框架 **AdMem**，融合**语义记忆**、**情景记忆**和**程序记忆**，并采用**短期记忆（STM）** 与**长期记忆（LTM）** 的双层设计。
- 采用**多智能体架构**：包括**行动者智能体**（与环境和用户交互）、**记忆智能体**（管理长期记忆的存储、检索和更新）、**评论者智能体**（生成程序记忆条目并标注奖励）。

### 关键技术细节
- **记忆分类**：
  - **语义记忆（MS）**：存储关于环境和任务的事实与通用知识。
  - **情景记忆（ME）**：存储过去交互事件的摘要。
  - **程序记忆（MP）**：存储从过去动作中提取的决策指导，包括上下文、动作、预期结果、实际结果和反思。
- **短期记忆管理**：采用类似计算机程序执行栈的结构，支持**计划任务（Plan for the new task）**、**添加子目标（Add sub-goal）**、**结束子目标（Conclude sub-goal）**、**思考（Think）**、**行动（Act）** 等工具，实现上下文压缩和清晰的范围管理。
- **程序记忆生成**：行动者智能体在执行动作时附带预期结果，评论者智能体后续收集观察结果，通过比较预期与实际生成奖励（0或1）和反思，并更新记忆条目的有效性参数 \(v_m\)。
- **长期记忆管理**：
  - **检索**：语义/情景记忆使用密集检索；程序记忆结合上下文相似度和记忆条目有效性评分（\(v_m \cdot \psi(c_t)^\top \psi(c_m)\)）。
  - **更新与剪枝**：基于奖励反馈，使用EM算法更新每个程序记忆条目的有效性参数\(v_m\)；当使用频率低于阈值\(\epsilon\)时删除；当两个条目经常同时被检索或同时不被检索时合并。
- **代理决策形式化**：将智能体决策建模为部分可观测马尔可夫决策过程（POMDP），记忆作为智能体状态的一部分。

## 6. 主要结论与发现

- AdMem在大多数领域显著提升了任务完成度和平均进度（例如AlfWorld从49.3%→63.4%，PDDL从33.3%→76.7%，BabyAI达到100%）。
- 在某些领域（如WebShop、Science World），AdMem的性能与ReAct持平或略有下降，但未出现如AWM的大幅下降，表明AdMem能较好地区分任务特定知识与可迁移知识，避免记忆干扰。
- 消融实验表明：
  - 短期规划能提升进展，但不跨任务共享信息，无法随任务数量增长而改进。
  - 长期程序记忆单独使用时会“烧录”早期性能退化（前几轮效果差），但后续epoch中逐渐提升，说明长期记忆需要一定的积累过程。
  - 完整AdMem结合两者，实现了长期自我进化能力。
- 多轮测试（Jericho三个epoch）中，AdMem从45%/0.5569提升至60%/0.6820，证明记忆系统具有持续改进的潜力。
