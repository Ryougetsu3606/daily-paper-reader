## 摘要
我们提出了\textbf{回声记忆}，这是一项关于行动条件的世界模型中记忆机制的受控研究。这些模型根据第一帧、文本提示和相机动作序列生成多段视频，但它们的核心失败往往是记忆而非局部图像合成：当相机离开再返回时，场景或显著物体可能会悄然改变。现有的记忆设计难以比较，因为改进与骨干网络、训练、检索和评估差异纠缠在一起。回声记忆固定了动作到视频的接口，仅改变生成器如何存储和读取历史。在共享的视频扩散骨干、优化器、相机动作表示、采样器和评估流程下，我们比较了原始上下文、基于压缩的记忆、具有不同读出路径的空间摘要以及状态空间循环。这个匹配矩阵区分了四个通常混淆的维度：\emph{容量}、\emph{压缩}、\emph{读出}和\emph{循环}。我们还通过三个分支协议评估记忆：重放质量、域内循环重访和开放域返回探测。这些分支经常不一致，表明重放保真度不足以作为记忆世界的代理。产生了三个发现。原始上下文是一个强大的容量基线，并且比改进重放指标更能显著提升开放域返回。紧凑性并不是容量的免费替代品：激进的空间和混合压缩记忆会丢失返回所需的关键证据。最后，块状状态空间循环是我们矩阵中最强的开放域返回机制，表明隐式记忆的结构与使用它的决策同样重要。这些结果为研究行动世界模型中的记忆提供了一个紧凑的协议，超越了孤立的重放指标。

## Abstract
We present \textbf{Echo-Memory}, a controlled study of memory mechanisms in action-conditioned world models. These models generate multi-segment videos from a first frame, text prompt, and camera-action sequence, but their central failure is often memory rather than local image synthesis: after the camera leaves and returns, the scene or salient object may silently change. Existing memory designs are hard to compare because gains are entangled with backbone, training, retrieval, and evaluation differences. Echo-Memory fixes the action-to-video interface and varies only how history is stored and read by the generator. Under a shared video diffusion backbone, optimizer, camera-action representation, sampler, and evaluation pipeline, we compare raw context, compression-based memory, spatial summaries with different read-out paths, and state-space recurrence. This matched matrix separates four otherwise conflated axes: \emph{capacity}, \emph{compression}, \emph{read-out}, and \emph{recurrence}. We also evaluate memory through a three-branch protocol: replay quality, in-domain loop revisit, and open-domain return probes. The branches routinely disagree, showing that replay fidelity is not a sufficient proxy for remembering a world. Three findings follow. Raw context is a strong capacity baseline and improves open-domain return far more than it improves replay metrics. Compactness is not a free substitute for capacity: aggressive spatial and hybrid-compression memories lose the salient evidence needed for return. Finally, block-wise state-space recurrence is the strongest open-domain return mechanism in our matrix, showing that the structure of implicit memory matters as much as the decision to use it. These results provide a compact protocol for studying memory in action world models beyond isolated replay metrics.

- **统一基准**：固定视频扩散Transformer（Video DiT）骨架、优化器、动作编码、采样器和评估流水线，仅改变记忆/上下文配置。

## 6. 论文的主要结论与发现

1. **回放质量 ≠ 记忆质量**：回放PSNR/SSIM和开放域VLM经常不一致（如空间记忆回放PSNR最高但开放域VLM仅6.0；块状SSM回放PSNR较低但开放域VLM达69.0）。**回放是健康信号，不是记忆得分**。

2. **原始上下文是强容量基线**：K=1→K=20使开放域VLM从12.25提升至58.63，而回放PSNR仅微增。**上下文主要帮助重返而非局部像素重建**。紧凑记忆应以K=20为基准，而非仅与I2V对比。

3. **紧凑性不等于语义记忆**：空间记忆在回放上强但在开放域返回上弱；长度缩减r=4（丢掉3/4时间位置）开放域VLM达43.25，优于保留所有位置的权重仅（22.38）。混合压缩（先缩减后权重）进一步下降至8.75，表明操作不当反而摧毁关键证据。

4. **存储与读出需分离评估**：空间记忆读出消融中，注入-无变体（保留令牌但不注入）反而获得最高回放PSNR（14.66），说明回放提升可能来自正则化而非有效记忆。专用交叉注意力使开放域VLM提高至17.12，但仍远低于上下文和块状SSM。

5. **结构化循环是关键**：块状状态空间循环（非可绕过状态）远超传统混合SSM（34.75 vs 69.0），表明隐式记忆的结构（如何整合状态）比“是否使用循环”更重要。

6. **三分支评估不可或缺**：任何单一指标（尤回放）都会误导选择。最佳设计需在**回放保真度、步进效率、重返泛化**三者间权衡。

## 7. 优点

- **严格的受控实验设计**：将记忆研究从“发明新模块”转向“在统一接口下变量分解”，避免了不同骨架、训练配方、采样器等因素的干扰。
- **创新的评估协议**：提出三分支评估（重放、域内重返、开放域重返），明确区分“跟随动作”和“记住世界”，并揭示二者非单调对应关系。
- **细致的消融分析**：分别隔离容量、压缩、读出、循环四个轴，特别是空间记忆的读出消融和压缩的混合消融，结论具有诊断价值。
- **开放域构建与VLM验证**：通过Qwen Edit生成身份锚定场景，并跨VLM判断者进行稳健性检验，增强语义评分的可信度。
- **实用导向**：指出不同下游任务（高保真回放 vs 交互式世界探索）需要不同的记忆设计，并给出具体轴向建议。

## 8. 不足与局限

- **单一数据集**：仅在Context-as-Memory数据集上训练，结果对数据集噪声、相机分布、场景类型敏感，泛化性未验证。
- **计算规模有限**：总步数仅5k，上下文长度最大20，未探索更大规模训练或多阶段课程的影响。
- **开放域VLM依赖**：虽然进行了跨判断者检验，但VLM评分仍可能受判断偏见、提示敏感性影响，缺乏大规模人类评估校准。
- **样本量小**：开放域每行仅8个案例（20×8池的一个子集），统计稳定性有限，小差异不可强推。
- **未探索训练时回归损失的重返感知**：当前训练仅监督目标帧，重访一致性仅在评估时测量，未设计专门的训练损失（如对比对齐、物体级保留损失）。
- **缺乏更长视野测试**：开放域仅45°往返，更复杂的360°循环或多站返回未评估。
