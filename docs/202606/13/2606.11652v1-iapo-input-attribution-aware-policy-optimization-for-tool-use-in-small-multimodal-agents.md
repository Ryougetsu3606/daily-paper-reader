## 摘要
本文研究强化学习（RL）方法以提升多模态小语言模型（SLM）智能体的工具调用能力。尽管现有工作探索了多种奖励设计来改进智能体的工具调用能力，但这些方法在SLM训练中面临固有局限，尤其是在多模态场景下。首先，许多现有方法通过与特定真实标签或预定义格式的精确匹配来评估工具使用的正确性。然而，这一假设通常不适用于多模态任务，因为多种工具使用路径可能都是有效的，且注释的工具轨迹通常不可用。其次，这种稀疏且脆弱的二进制奖励几乎无法指导如何改进底层决策过程，使得多模态SLM难以从中学习。为了解决这些问题，我们提出输入归因感知策略优化（IAPO），这是一种RL算法，通过使模型在输入组件上的归因与更强的教师模型对齐，来改进多模态SLM中的工具使用。在Qwen2.5-VL-3B上的实验表明，与现有的视觉工具使用方法相比，所提方法通过帮助模型关注最相关的输入证据，在六个测试集上将视觉问答准确率平均提高了3%。

## Abstract
This paper investigates reinforcement learning (RL) methods for improving tool-calling capabilities in multimodal small language model (SLM) agents. While existing works have explored various reward designs to improve agentic tool-calling ability, these approaches face inherent limitations for SLM training, especially under multimodal scenarios. First, many existing methods evaluate tool use correctness through exact matching against certain ground-truth or predefined formats. However, this assumption is often unsuitable for multimodal tasks, where multiple tool use paths may be valid and annotated tool trajectories are typically unavailable. Second, such sparse and brittle binary rewards provide little guidance on how to improve the underlying decision process, making them particularly difficult for multimodal SLM to learn from. To address these issues, we propose Input Attribution-Aware Policy Optimization (IAPO), an RL algorithm for improving tool use in multimodal SLM by aligning the model's attribution across input components with that of a stronger teacher. Experiments on Qwen2.5-VL-3B show that the proposed method improves visual question answering accuracy by an average of 3% across six test sets compared with existing visual tool use work, by helping the model attend to the most relevant input evidence.

## 1. 论文的核心问题与整体含义（研究动机和背景）
现有强化学习方法（如GRPO）用于训练多模态小语言模型（SLM）的工具调用能力时，存在两个主要局限：
- **奖励设计不匹配**：许多方法依赖精确匹配（exact matching）来评估工具使用正确性，但在多模态任务中，多个有效工具路径并存，且真实工具轨迹难以获得。
- **奖励信号稀疏**：稀疏的二元奖励（正确=1，错误=0）无法提供中间过程的学习指导，尤其对参数较小的SLM难以有效优化。

这些局限导致SLM在多模态工具使用场景中决策质量低下。论文提出输入归因感知策略优化（IAPO），通过将学生模型对输入组件的归因（attribution）与更强教师模型对齐，提供密集的过程级监督信号，从而改善小模型的工具调用行为。

## 2. 论文提出的方法论
### 核心思想
利用集成梯度（Integrated Gradients, IG）量化输入中不同语义块（如纯文本上下文、用户查询、图像占位符、工具定义）对生成工具调用序列的贡献，然后最小化学生与教师模型归因分布的KL散度，作为额外惩罚项，与结果奖励（outcome reward）共同构成最终奖励，再通过GRPO框架优化策略。

### 关键技术细节
1. **输入分块**：将提示词划分为有序语义块 \( x = [b_1, \dots, b_K] \)，类型包括纯文本上下文（P）、用户查询（R）、图像（I）、工具定义（T）。
2. **集成梯度得分**：
   - 对输入令牌嵌入 \( e_i \)，计算工具调用序列对数概率的梯度 \( g_i = \partial s(E) / \partial e_i \)。
   - 令牌级归因：\( o_i = \| g_i \odot e_i \|_2 \)。
   - 块级归因：\( O_k = \sum_{i \in \Omega_k} o_i \)（\( \Omega_k \) 是块 \( b_k \) 对应的令牌索引集）。
3. **归因对齐惩罚**：
   - 对学生和教师分别归一化块级归因为分布 \( \hat{a} \)、\( \hat{b} \)。
   - 计算KL散度 \( P_A = D_{KL}(\hat{b} \|\hat{a}) \)。
4. **最终奖励**：\( R_{\text{IAPO}} = R_{\text{out}} - \lambda P_A \)，其中 \( R_{\text{out}} \) 为结果奖励（正确=1，错误=0），\( \lambda \) 控制惩罚强度。
5. **训练流程**：
   - 第一阶段：冷启动GRPO，赋予模型基本工具调用能力。
   - 第二阶段：使用上述归因奖励进行IAPO训练。
   - 教师模型：由7B参数的Qwen2.5-VL-7B-Instruct通过标准GRPO预训练获得。

## 6. 论文的主要结论与发现
- IG得分与工具调用正确性强相关：学生模型对正确工具定义的归因越高，工具选择越准确。
- IAPO在六个测试集上平均准确率高于VTool-R1约3%，在泛化基准CharXiv上优势更明显。
- 与TRM和ToRL相比，IAPO能提供更丰富的学习信号，不依赖匹配唯一正确路径。
- 训练动态显示IAPO能更稳定地提升工具调用率（接近1.0）和成功率（后期接近100%），而VTool-R1不稳定。
- 教师模型的选择：同一模型族（如7B+GRPO）的教师效果优于更先进但族不同的模型（如Qwen3-VL-8B）。

## 7. 优点
- **创新奖励设计**：首次利用输入归因作为密集、连续的过程级监督信号，克服了稀疏二元奖励的局限性。
- **有效迁移教师知识**：通过归因分布对齐而非预测输出对齐，避免硬标签匹配，更适合多模态工具路径多样性。
- **实验验证充分**：包含案例研究、消融分析、训练动态可视化，逻辑自洽。
- **适用于小模型**：在3B参数上取得显著提升，实用性强。
